# Share Projects

Add, Edit, Delete, View Project links.

## Stack

- React
- Express + Node
- Mongoose

## Features

- Add Project
- Delete Project
- Update Project
- Get Project(s)

## Plugins

- `babel-cli`
- `babel-preset-env`
- `babel-preset-stage-0`
- `cors`
- `nodemon`

## Demo

- Use `npm run dev` in `server` folder to run server
- Use `npm run start` in `client` folder to run client
- Make sure MongoDB database is running on the system
