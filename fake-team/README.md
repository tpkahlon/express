# Fake Team

## Features

- Handles all CRUD operations for REST API
- `GET`, `PUT`, `DELETE`, `POST` HTTP calls are addressed
- Input validation is handled via `Joi/HAPI` validation

## Testing

- Use `POSTMAN`
- Install it via `brew cask install postman`
